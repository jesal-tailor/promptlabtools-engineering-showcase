# Terraform Scaffold

This folder contains a documented Terraform scaffold for future productionisation planning.

It is intentionally not required for the public-safe demo.

## Intended Use

- Discuss cloud architecture in interviews.
- Show where app hosting, runtime services, logs, secrets, queues, and databases could live.
- Keep production infrastructure out of the public repo.

## Not Intended For

- Creating real AWS resources by default.
- Storing credentials.
- Deploying the public showcase.
- Connecting to production PromptLabTools systems.

## Safety

The AWS scaffold uses placeholder names and disables placeholder resources by default.
