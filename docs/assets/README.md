## Safety Checklist

Before replacing these images, confirm that screenshots include only local mock data and do not reveal:

- Secrets or environment variables.
- Customer data.
- Real analytics.
- Real email addresses beyond `.test` examples.
- Browser account state.
- Production PromptLabTools prompts, automations, or integrations.

The current image set is intended for public repository use.
